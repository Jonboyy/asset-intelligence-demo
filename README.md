# Asset Intelligence Assistant

A full-stack demo project that showcases:

- React + Vite frontend
- FastAPI backend
- PostgreSQL database
- AI-assisted analytics workflow for enterprise asset management
- safe backend validation between AI output and database queries

## Project Goal

The application simulates an internal assistant for IT, operations, and asset management teams. Users can ask business questions about company assets, assignments, maintenance history, warranty risk, and software license utilization.

The AI assistant interprets the request, produces a structured query intent, and sends it to the backend. The backend validates the request, retrieves data from PostgreSQL, and returns structured results for frontend rendering.

## Planned Stack

- Frontend: React, Vite, TypeScript
- Backend: FastAPI, Python
- Database: PostgreSQL
- AI: OpenAI API
- Styling: Tailwind CSS
- Charts: Recharts

## Planned Features

- authentication and role-based access
- chat-style assistant interface
- structured AI task parsing
- safe backend query validation
- asset lifecycle analytics
- maintenance cost analysis
- offboarding audit checks
- software license utilization reporting
- table and chart-based result rendering

## Demo Domain

The demo company is fictional:

**NordAxis Mobility Group**

A fictional multinational industrial and mobility technology company used to generate realistic asset and operations data for the application.
